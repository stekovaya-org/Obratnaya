# This is readme for [index.toml](index.toml)
index.toml format:
```toml
[name]
name = "name"
code = "code"
output = "output"
```
### Options
 - name
   - **Required**, we use this option for display `-----name`
 - code
   - **Required**, we use this option for run code
 - output
   - **Required**, we use this option for check `result of code === output`
